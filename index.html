How to Play

Type commands in the editor to move the player ● to the goal ◎ on the 6x6 grid.

Rules: Put one command per line. Hitting a wall or stepping off the grid will stop your program. Press Reset and try again. Finish all 6 levels, then translate your code into binary, just like a real compiler does. You can also repeat a command using COMMAND(N) (like MOVE_RIGHT(3)).

From Source Code to Machine Code

Every command you typed, like MOVE_RIGHT, is source code: text humans read and write. Before a CPU can run it, a compiler translates it into binary code made of 1s and 0s. For example, MOVE_RIGHT becomes 011. The CPU reads direct binary to control its circuits. This process of changing text to binary to electricity is how every computer program runs.
